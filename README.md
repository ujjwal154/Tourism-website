# Tourism-website

# 🏞️ Himachal & Uttarakhand Tourism Website

A beautifully designed single-page Flask-based travel website showcasing the natural beauty and cultural events of **Himachal Pradesh** and **Uttarakhand**. Includes dedicated pages for festivals, cottages, Ladakh tours, and contact information.

---


---

## ✨ Features

- 🏔️ Dedicated pages for Himachal, Uttarakhand, and Ladakh
- 🎉 Festivals and Events Page
- 🏡 Cottage Stay Showcase
- 📞 Contact Page
- ⚙️ Flask-based Routing with Jinja2 Templates
- 💅 Fully responsive and customizable HTML/CSS UI

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Bootstrap, Jinja2 Templates
- **Backend**: Python 3, Flask
- **Deployment**: Render / Replit / GitHub Pages (static front)

## 🚀 Getting Started Locally

### 1. Clone the Repository

```bash
git clone https://github.com/ujjwal154/Tourism-website/tree/main
cd tourism-website
```

### 2. Install Dependencies

```bash
pip install -r requirement.txt
```

### 3. Run the Flask App

```bash
python app.py
```

App runs on: `http://127.0.0.1:5000`
